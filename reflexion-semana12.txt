Durante este curso aprendí que la tecnología puede ser una herramienta muy importante para fortalecer la participación ciudadana y la organización comunitaria. También comprendí cómo integrar bases de datos, servidores y automatizaciones para desarrollar soluciones útiles. Aprendí la importancia de proteger la información personal mediante buenas prácticas de seguridad. Conocí el uso de Git y GitHub para controlar versiones de los proyectos. Además, entendí cómo Node.js permite crear aplicaciones web sencillas y funcionales. La automatización con herramientas como n8n o Telegram facilita la comunicación entre los usuarios. También comprendí la utilidad de generar documentos PDF de forma automática para apoyar procesos comunitarios. En conclusión, este curso me permitió adquirir conocimientos prácticos para desarrollar aplicaciones tecnológicas orientadas al beneficio de la comunidad.
